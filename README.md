# middlewear とは

リクエストがきた時に順次 app.use で読み込まれるものを適用して行きなさいという指令のこと。

`next();`を付け忘れると、次の処理に移れないので注意。

# コマンドライン

mkdir：ディレクトリ作成
touch：空ファイル作成
cd ..：１つ上のディレクトリに移動

# filename / dirname

`__filename`: 現在実行中のモジュールの絶対パスを取得する
`__dirname`: 現在実行中のモジュールが格納されているディレクトリの絶対パスを取得する
